# Video Library - видео-библиотека

### Основные операции в системе:
- Поиск и фильтрация фильмов
- Просмотр информации по конкретному фильму (год выпуска, жанр, члены съемочной группы и т.д.)
- Добавление ревью к фильму
- Добавление нового фильма в библиотеку (включая картинку, описание, года выпуска и членов съемочной группы)
- Добавление новых членов съемочных групп (новые актеры, композиторы и т.д.)

### Действия в системе для пользователя с ролью admin также включают в себя:
- Поиск и фильтрация по пользователям системы (по email пользователя)
- Просмотр общей информации о каждом пользователе (имя, список всех ревью ко всем фильмам)
